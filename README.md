# 🔐 KeySmith

**Crafting Stronger Passwords.**

KeySmith is a modern, fast, and secure password generator that helps users create strong passwords in seconds. It combines traditional random password generation with an intelligent **Word-Based Password Generator**, allowing users to transform familiar words into complex, memorable, and secure passwords.

## 🌐 Live Demo

👉 **https://key-smith-security-agent.vercel.app/**

---

## ✨ Features

### 🔑 Smart Password Generator
- Generate secure random passwords
- Adjustable password length
- Include or exclude:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols

### 🧠 Word-Based Password Generator
Generate strong passwords using a word provided by the user.

**Example**

Input:

```text
password
```

Generated:

```text
P@ssw0rd#92
P@55w0rd_X7
P4ssw0rd!2026
P@ssW0rd$Secure
```

## 🚀 Why KeySmith?

Most password generators only create random strings.

KeySmith goes one step further by allowing users to generate secure passwords around words users already remember, making passwords both **strong** and **memorable**.

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/keysmith.git
```

Navigate to the project:

```bash
cd keysmith
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
